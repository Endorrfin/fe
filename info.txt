IntelliJ IDEA Guide

Ctrl + E			последние открытые файлы
Ctrl + Shift + E	последние редактированные файлы
Alt + Enter		    показать быстрые действия, исправить код
Ctrl + F12		    список методов текущего класса
Ctrl + Alt + L		автоматически отформатировать код
Alt + Insert		сгенерировать код (конструктор, геттер, сеттер, equals, hashCode)
Shift + F6		    переименовать (класс, метод, поле, переменную), включая все использования и референсы
Ctrl + I			переопределить метод
Alt + J 			имплементировать метод
Ctrl + /			закомментировать однострочным комментарием
Ctrl + Shift + /	закомментировать многострочным комментарием
Shift + Alt + Insert	переключить режим редактирования (обычный/колонка)
Shift + Shift		поиск везде
Ctrl + `			быстрое переключение текущей схемы
Ctrl + Shift + V	вставить фрагмент кода из буфера обмена
Alt + Inc		    генерация кода
Ctrl + Alt + S		настройки IDEA
Ctrl + Alt + Shift + S	настройки проекта
Shift + F10		    запуск
Ctrl + [ or ]		перейти в конец / начало блока кода
Ctrl + B			перейти к объявлению
Ctrl + Shift + F7	подсветить использования в файле
Alt + Enter		    показать предлагаемое исправление
Alt + Shift + C		посмотреть последние изменения
Alt + Shift + mouse	поставить курсор в несколько мест
Ctrl + / -			развернуть / свернуть блок кода
Ctrl + Shift + +	развернуть/свернуть всё
Alt + F8			evaluate expression окно и прогоняете все варианты PROFIT
Ctrl + P			показывает вам список принимаемых методом параметров
Ctrl + Q			во вложенном окне покажет документацию к методу
Ctrl + B			переход в исходный код класса\метода либо переход к объявлению переменной
Alt + F7			покажет, где используется переменная\метод, альтернатива Ctrl+F
Ctrl + Shift + U	смена регистра у слова
Ctrl + Shift + Space	умный комплишен, предлагает вам варианты подстановки значений с учетом контекста
Ctrl + Shift + A		поиск действия. Если вдруг забыли hot keys для действия, но помните его имя
Alt + влево/вправо	Перемещение между вкладками
F12			        Открыть последнее закрытое окно
Ctr + Shift + I		Показать реализацию метода или класса
Ctr + B		        Перейти к объявлению переменной, класса, метода
Ctr + Space		    Полный автокомплит
Alt + J/Shift + Alt + J для создания нескольких кареток. Alt + J выбирает следующее вхождение текущего выделенного текста и добавляет другую каретку.
